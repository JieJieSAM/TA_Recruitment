# Supporting Materials (Iteration 1)

This document provides evidence templates and initial content for the first assessment appendix.

## 1. Interview Guide

### Target Participants

1. TA representative
2. Module Organiser representative
3. Admin/coordination representative

### Core Questions

1. What information is most often missing in current TA applications?
2. Which steps currently consume the most manual effort?
3. What status states are necessary for decision tracking?
4. What workload indicators are needed for allocation decisions?
5. What are common errors in current form/spreadsheet process?

## 2. Questionnaire Template (Quick Form)

### Suggested Questions

1. Which feature is most urgent? (Profile / Job posting / Status tracking / Workload view)
2. Preferred status labels? (Submitted / Shortlisted / Rejected / Offered)
3. Is CV upload or CV link/reference sufficient for first release?
4. What is acceptable maximum steps for TA to complete one application?
